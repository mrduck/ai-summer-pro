## 详细说明

```
AI 网页总结助手是一款基于 DeepSeek AI 的智能网页内容总结工具，帮助你快速提取网页核心信息，提升阅读效率。

【核心功能】
• 一键总结：打开任意网页，点击扩展图标即可自动提取页面内容并生成 AI 总结
• 三种模式：要点摘要（快速浏览）、段落总结（结构清晰）、详细分析（深度解读）
• 段落引用：每条总结观点均标注来源段落，点击即可跳转至原文位置
• 历史记录：自动保存总结历史，支持搜索和回顾，随时查看过往总结
• 多语言支持：界面支持中文、English、日本語，自动检测浏览器语言
• 免费额度：每日免费 5 次总结，Pro 用户无限制使用

【使用场景】
• 阅读长篇文章时快速了解核心观点
• 学术研究时高效提取论文关键信息
• 技术文档学习时把握重点内容
• 新闻资讯浏览时节省阅读时间

【隐私说明】
• 本扩展仅在用户主动点击总结时发送当前页面内容至 AI 服务
• 不会自动采集或上传任何浏览数据
• 不追踪用户行为，不收集个人信息
```

## 权限说明（审核时填写）

| 权限 | 用途说明 |
|------|------|
| `activeTab` | 获取当前标签页的网页内容用于总结 |
| `scripting` | 通过 Readability 算法提取页面正文，排除广告和导航 |
| `storage` | 本地保存设备标识和用户偏好设置 |
| `tabs` | 检测标签页切换，自动刷新侧边栏内容 |
| `sidePanel` | 在浏览器侧边栏展示总结界面 |
| `contextMenus` | 右键菜单快捷操作 |
| `<all_urls>` | 读取所有网页内容以提供总结服务，不采集或上传任何数据 |

## 提交后注意事项

1. **审核时间**：通常 1-3 个工作日
2. **审核通过后**：
   - 记录 Google 分配的永久扩展 ID
   - 将 `key` 字段加回 `manifest.json`（用于后续开发时固定 ID）
   - 在 ExtensionPay 后台更新扩展 ID（如需）
3. **版本更新**：每次更新需修改 `manifest.json` 中的 `version` 字段，重新打包 ZIP 上传
