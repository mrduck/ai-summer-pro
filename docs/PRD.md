## 一、产品概述

### 1.1 产品定位

一款 Chrome 浏览器扩展，帮助用户在浏览网页时一键生成 AI 摘要、结构化要点和思维导图，提升信息获取效率。

### 1.2 目标用户

- **重度阅读者**：每天浏览大量文章，需要快速筛选信息
- **知识工作者**：程序员、产品经理、研究员，需要整理和回顾阅读内容
- **学生/学习者**：阅读论文、教程、文档，需要提炼要点

### 1.3 核心价值

> "浏览网页 → 一键总结 → 结构化知识沉淀"

- 节省阅读长文的时间（3 秒获取核心观点）
- 将碎片化浏览转化为可检索的知识库
- 脑图形式让知识结构一目了然

## 二、功能需求

### 2.1 MVP 功能（第一版）

#### F1 - 页面一键总结

| 字段 | 描述 |
|------|------|
| 触发方式 | 点击扩展图标 → Side Panel 弹出 → 点击「总结本页」按钮 |
| 输出模式 | ① 一句话总结 ② 段落摘要 ③ 结构化要点列表 |
| 处理流程 | 提取网页正文 → 发送 AI API → 流式返回结果 |
| 加载体验 | 打字机效果，逐字展示结果 |

#### F2 - 选中文字总结

| 字段 | 描述 |
|------|------|
| 触发方式 | 选中网页文字 → 右键菜单 → 「AI 总结选中内容」 |
| 输出模式 | 同 F1 |
| 优先级 | P1（核心差异化） |

#### F3 - 多语言输出

| 字段 | 描述 |
|------|------|
| 输入 | 自动识别网页语言 |
| 输出 | 用户可选择中文 / English / 日本語 输出 |
| 默认 | 中文 |

#### F4 - 历史记录

| 字段 | 描述 |
|------|------|
| 存储内容 | 原文 URL、标题、摘要结果、生成时间 |
| 检索方式 | 关键词搜索、按时间排序 |
| 操作 | 查看详情、删除、重新生成 |

#### F5 - 设置面板

| 字段 | 描述 |
|------|------|
| 可配置项 | 默认输出语言、默认输出模式、API Key（可选） |
| 位置 | Side Panel 底部齿轮图标 |

### 2.2 第二版功能（V2，MVP 验证后）

- **F6 - 智能脑图生成**：长文自动生成 Markdown 思维导图（Mermaid 渲染）
- **F7 - 深度阅读模式**：一键提取正文（类 Reader View），去掉广告和干扰
- **F8 - 导出功能**：摘要导出为 Markdown / PDF
- **F9 - 跨文章关联**：自动发现多篇文章之间的关联观点

### 2.3 不做的事（明确边界）

- 不支持视频/音频总结（待 V3 评估）
- 不做社交分享（专注工具属性）
- 不做浏览器首页/新标签页改造

## 三、用户流程

### 3.1 核心流程

```
用户浏览网页
    │
    ├─→ 点击扩展图标 → 打开 Side Panel
    │       │
    │       ├─→ 点击「总结本页」
    │       │       │
    │       │       └─→ 提取正文 → 调用 AI → 打字机展示结果
    │       │
    │       └─→ 切换输出模式（一句话 / 段落 / 要点）
    │
    └─→ 选中文字 → 右键 → 「AI 总结选中内容」
            │
            └─→ 弹出结果浮层
```

### 3.2 历史回顾流程

```
打开 Side Panel → 点击「历史」Tab → 搜索/浏览 → 点击查看详情
```

## 四、非功能需求

| 需求 | 指标 |
|------|------|
| 性能 | 正文提取 < 1s，AI 首字响应 < 3s |
| 可用性 | 核心操作不超过 2 次点击 |
| 兼容性 | Chrome 110+，Manifest V3 |
| 安全 | API Key 加密存储，HTTPS 通信 |
| 免费配额 | 未登录用户每天 5 次总结 |
